TEMA 2 - Grundlæggende Web

Opgaven: Responsive site 2

I de første uger af dette tema fik vi i grupper udleveret en stilart hver. Min gruppe fik surrealisme. Siden skulle vi da, ud fra den givne stilart, udarbejde vores eget site om denne. Fokus var at lære css samt væsentlige web konventioner. I css var der især fokus på mediaqueries og flexbox. Flexbox manipulere med positionen af de forskellige elementer, mens mediequeries gør siden responsiv. Begge dele er nødvendige for at kunne lave websites.
